Help / About

TabTogether lets you send tabs to groups of devices instantly.
To use:

  • Use the popup to send the current tab to any group.
  • Manage device name, groups, and subscriptions here in settings.
  • Rename groups by clicking their name.
  • Use the test notification to check if notifications work.
  • Dark mode can be toggled manually or follows your system preference.
  • Leaving or deleting a group is permanent and affects all devices.

For more help, visit the project page.
